# Bankly


A mobile responsive banking application using React, styled components, and smooth scrolling. 
#####(Created through a @briancodex tutorial)

## Languages:
* Javascript
* HTML
* CSS (through styled components)

## Technologies:
* React
* React Router Dom
* Styled Components
* Smooth Scrolling

## Deployment:
deftone1.github.io/bankly/
